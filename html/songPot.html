SongPot

Una aplicación móvil social para descubrir y compartir música. Conecta con otros usuarios, explora recomendaciones personalizadas y amplía tu biblioteca musical con canciones que realmente importan.

Detalles del Proyecto

SongPot es una aplicación Android desarrollada en Kotlin que permite a los usuarios compartir y descubrir nueva música. En lugar de un sistema de recomendación complejo, la app se centra en mostrar canciones que aún no han sido escuchadas por el usuario, fomentando así un descubrimiento musical auténtico y fresco.

Características principales:

  • Descubrimiento de canciones nuevas no escuchadas previamente
  • Integración con APIs de Spotify y YouTube Music para reproducción
  • Perfiles de usuario con listas de favoritos
  • Sistema de seguimiento entre usuarios con gustos similares
  • Búsqueda avanzada por género, artista, año y estado de ánimo
  • Notificaciones push cuando tus contactos comparten música

El backend fue desarrollado con Spring Boot y MongoDB, lo que permitió gestionar de forma eficiente los datos de usuarios y canciones. También se integró Firebase para la autenticación y notificaciones en tiempo real, junto con Retrofit2 para el consumo de APIs externas. La aplicación sigue una arquitectura MVVM con principios de Clean Architecture, garantizando un código modular y escalable.
